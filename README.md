## 📖 О проекте

"Stellar Station" — это многопользовательский симулятор космической станции. Игроки берут на себя роли членов экипажа на борту огромного космического объекта, где хаос может начаться в любой момент из-за предателей, инопланетных угроз или банальной халатности.

Мы переосмысливаем классический геймплей, добавляя улучшенную систему освещения, физику предметов и инструменты для моддинга на базе Unity.

---

## 🎮 Как начать играть

На данный момент проект находится в стадии активной альфа-разработки. Публичные серверы откроются позже.

Если вы хотите собрать билд локально:
1.  Клонируйте репозиторий: `git clone https://github.com/R-D-Station/StellarStation/tree/main`
2.  Откройте папку проекта в Unity Hub (версия `6000.3.1f11`).
3.  Запустите сервер (откройте терминал в удобном для Вас месте и выполните команды):
    * `dotnet new console -n TestGameServer`
    * `cd TestGameServer`
    * `notepad Program.cs`
    * Скопируйте содержимое `Assets/Scripts/Server/TestServer.cs` в `Program.cs`
    * Запустите сервер командой `dotnet run`
4.  Дождитесь импорта ассетов и запустите сцену `TestMap` в папке `Assets/Scenes/`.
